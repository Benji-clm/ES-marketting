Technical Specifications

Hardware

  • Raspberry Pi v2.1 camera module
  • I2C-based temperature and humidity sensor - Adafruit Si7021
  • Air quality sensor (eCO2, TVOC) - Sparkfun CCS811

Communication

  • End-to-end encrypted communication (TLS 1.3)
  • Mutual TLS (mTLS) device authentication
  • Per-device identity verification
  • Backend isolated behind secure reverse proxy
  • Minute-level telemetry updates

Backend & Processing

  • EC2-hosted server-side processing
  • YOLOv26-Nano or YOLOv26-Medium models for people counting
  • Historical data storage and analytics
  • Predictive analytics engine

Features

  • Real-time occupancy monitoring
  • Environmental condition tracking
  • Predictive busyness forecasting
  • Privacy-preserving design - no Database, no data collection
